制御構文の学習

1. if文

{% raw %}{% if show_message %}
    メッセージを表示
{% else %}
    メッセージはありません
{% endif %}{% endraw %}
結果: {% if show_message %}

メッセージを表示

{% else %}

メッセージはありません

{% endif %}

2. for文

{% raw %}{% for user in users %}
    {{ user.name }} ({{ user.age }}歳)
{% endfor %}{% endraw %}
結果:
    {% for user in users %}
  • {{ user.name }} ({{ user.age }}歳)
    • {% endfor %}

3. loop変数の使用

{% raw %}{% for n in numbers %}
    {{ loop.index }}番目: {{ n }}
{% endfor %}{% endraw %}
結果:
    {% for n in numbers %}
  • {{ loop.index }}番目: {{ n }}
    • {% endfor %}
