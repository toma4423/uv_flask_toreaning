フィルターの学習

1. 文字列フィルター

{{ '{{ text|capitalize }}' }}
{{ '{{ text|upper }}' }}
{{ '{{ text|title }}' }}
結果:
  • capitalize: {{ text|capitalize }}
  • upper: {{ text|upper }}
  • title: {{ text|title }}

2. HTMLエスケープ

{{ '{{ html_text|escape }}' }}
{{ '{{ html_text|safe }}' }}
結果:

escape: {{ html_text|escape }}

safe: {{ html_text|safe }}

3. リストフィルター

{{ '{{ numbers|join(", ") }}' }}
{{ '{{ numbers|length }}' }}
{{ '{{ numbers|first }}' }}
{{ '{{ numbers|last }}' }}
結果:
  • join: {{ numbers|join(", ") }}
  • length: {{ numbers|length }}
  • first: {{ numbers|first }}
  • last: {{ numbers|last }}
